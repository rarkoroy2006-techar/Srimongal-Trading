# Srimongal Trading View

A fullscreen crypto trading chart website powered by the TradingView widget.  
Includes a watermark and quick controls for symbols, intervals, and fullscreen toggle.

---

## Run Locally
1. Open `index.html` in your browser.

## Deploy to GitHub Pages
1. Push repo to GitHub.
2. Settings → Pages → Branch: main, Folder: root.
3. Save → Live site link will appear.

## Notes
- Default symbol = BTC/USDT.
- TradingView branding cannot be removed.
